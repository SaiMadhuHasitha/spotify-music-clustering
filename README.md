# Spotify Music Clustering 🎧

## Project Overview

This project clusters Spotify songs based on audio characteristics using the K-Means clustering algorithm.

## Features Used

* Danceability
* Energy
* Tempo
* Loudness
* Valence

## Methodology

1. Data preprocessing
2. Feature normalization
3. KMeans clustering
4. PCA visualization
5. t-SNE visualization
6. Cluster analysis

## Dataset

Dataset was obtained from Kaggle and is not included in this repository due to size limitations.
https://www.kaggle.com/datasets/zaheenhamidani/ultimate-spotify-tracks-db
